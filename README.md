# AI Fake Profile Detection System

AI Fake Profile Detection System is a modern web-based dashboard designed to identify fake and suspicious social media profiles. The project demonstrates how machine learning and behavioral analysis can be used to evaluate profile authenticity through multiple indicators such as account age, follower patterns, posting behavior, profile information, and risk scoring.

## Features

* Interactive fake profile scanner
* Risk score analysis dashboard
* Machine learning model comparison
* Dataset visualization and statistics
* System architecture overview
* Responsive and modern UI design

## Technologies Used

* HTML
* CSS
* JavaScript
* Machine Learning Concepts
* Data Visualization
