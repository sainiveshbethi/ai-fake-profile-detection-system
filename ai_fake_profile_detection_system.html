Project Timeline
Phase 1 — Data Collection & EDA
Scraped 500K+ social media profiles from publicly available datasets (Kaggle, Twitter API, LinkedIn scraper). Performed exploratory data analysis to understand class imbalance — genuine:fake ratio ≈ 8:2.
Week 1–2
Phase 2 — Feature Engineering
Extracted 42 behavioural, structural, and NLP-based features: account age, posting frequency, bio sentiment, username entropy, follower/following ratio, image face detection, and more.
Week 3–4
Phase 3 — Model Training & Evaluation
Trained 6 ML/DL models: Logistic Regression, Random Forest, XGBoost, SVM, LSTM (on text sequences), and a CNN for profile image analysis. Used SMOTE for class-imbalance handling.
Week 5–7
Phase 4 — Ensemble & Deployment
Built a stacked ensemble achieving 97.4% accuracy. Deployed via FastAPI backend with React frontend. Integrated real-time scanning pipeline with Redis queue for batch processing.
Week 8–10
Phase 5 — Monitoring & Iteration
Implemented MLflow for model tracking, Prometheus for system metrics. Drift detection alerts when accuracy drops below 95%. Model retraining triggered weekly with new labelled data.
Ongoing
